How it works

Three steps from a sentence to a part on the build plate.

01

Describe it

Write JavaScript or OpenSCAD, or just tell the agent what you need. Code returns a Manifold — nothing else to learn.

02

Render live

It evaluates in a sandbox and renders in a real-time viewport. Geometry stats verify the part is watertight and single-bodied.

03

Export & print

Download print-ready GLB, STL, 3MF, or exact STEP. Sessions and versions persist locally in the browser.
