## Key decisions

**Eval-driven development with a tiered, pluggable judge.** Built `eval:figures`
(`scripts/eval-figures.mjs` + `scripts/cli/judge.mjs`) — a visual regression
suite for the figure library where the "assertion" is a vision judge. A case
pins a reference (target look) + rubric (anatomical checklist); the harness
builds with the *current* `F.*` primitives, renders matched-angle tiles (reusing
`scripts/cli/preview.mjs`), runs printability gates, judges vs the reference, and
checks the score against a committed `baseline.json` so a primitive fix can't
silently regress the rest of the corpus.

**Three judges, tiered by cost — the crux of the user's budget constraint.**
- `pixel` (default): free, offline structural-similarity sentinel. Not an anatomy
  judge — proves the loop and catches silhouette drift with no API key. Chosen so
  the whole loop is runnable + provable in the remote container (which has no
  `gemini` CLI/key).
- `human`: free offline anchor; emits a contact sheet + rubric-populated verdict
  template, read back with `--record`. Keeps the cheap judges honest.
- `gemini`: the real semantic judge; shells to the `gemini` CLI on the *user's*
  machine (separate quota from Claude Max). Returns a part-level checklist + a
  suggested geometry fix per item — never a bare scalar (that's what made the
  manual "hand it a photo" workflow work).

**Spend is instrumented + budget-capped** (`--budget`) so looping cost is visible
— directly answering "I want to see the impact on AI spend/usage."

**Builds on existing infra, doesn't reinvent it.** Discovered `scripts/
generate-reference-images.cjs` (Gemini+nanobanana photo→multi-angle refs) and the
Images-tab attach flow already exist; documented that they feed the reference
corpus rather than building a parallel path.

**Proven end-to-end in-container with the pixel judge:** pinned reference → 100/100
vs itself → committed baseline → perturbed shoulders → score dropped → regression
flagged → harness exits 1 (CI-gateable). Gates also demonstrated (genus caught a
pathological perturbation). The 2-point pixel sensitivity to a *local* region is
itself the argument for the semantic region-aware judge being the real product.

**Scope:** v0 harness + one `shoulders` case. Tracking issue #827 carries the
remaining deliverables (corpus expansion to ~25 cases, the browser-faithful
component/paint oracle from #697, the iteration controller in `model-sculpt`).
