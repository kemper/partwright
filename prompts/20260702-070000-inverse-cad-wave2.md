## Pattern the subagents converged on

Two techniques dominate:
1. **STL slice-and-trace** — parse target STL, slice at a mid-height Z inside
   the flat prismatic wall, walk boundary edges into an ordered polygon,
   simplify with Douglas-Peucker, feed to `geom.fromPoints(...).extrude(depth, 0, 0, [1, 1])`.
   Ankle nailed 0.16 mm this way.
2. **Cross-section probe + primitives** — ray-cast the target STL along Y/Z
   at a grid of X, read component extents empirically, then build in that
   absolute frame with simple primitives (`Manifold.sphere`, `Manifold.cube`,
   `CrossSection.offset(r, 'Round')`, `extrude(_, 0, 0, [1, 1])`). Thigh,
   upper arm, chest, hip/shoulder all used this.

**All subagents independently rediscovered**:
- Targets are Z-flat (Z ∈ [0, thickness]), NEVER centered on origin.
- `CrossSection.extrude` `scaleTop` scalar `1` silently produces a pyramid;
  must be `[1, 1]`.
- Y-offset is always significant — always call `meshBBox` on the parsed STL
  first, before writing a single line of geometry code.
