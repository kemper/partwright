## Key decisions

**Gates fail before writing OUT.** A failed `--max-genus` / `--require-labels`
check exits 1 *without* writing the entry file, so a regressed bake can't be
committed by accident — that's the whole point of making them exit-code gates.
The label inventory (`listLabels`) is now collected on every bake, not just
painted ones, so `--require-labels` also works for unpainted entries.

**`--require-labels` checks three failure modes per label**: missing from the
run, present in `lostLabels` (label stripped by the engine), or in
`paintByLabels().failed` (resolved to 0 triangles — the "buried eyes" mode
that bit two figures last round).

**Palettes: committed files AND a recovery flag, sharing one helper.** The
durable home is `public/catalog/palettes/<id>.json` (flat `{label: hex}`,
exactly the `--palette-file` format), generated by the new
`extract-catalog-palettes.cjs` from each entry's newest version with
`byLabel` colorRegions — committed palettes are the source for re-bakes,
and the extractor is the documented reconstruction made mechanical.
`--palette-from-existing <entry.json>` covers entries never committed there.
Both paths use `scripts/lib/catalog-palette.cjs` (no copy-paste drift).
Extraction found 13 palettes (10 figures + orrery, pocket watch, smiley
cactus); 106 entries have code-baked colors and correctly get no file.
`lint:catalog` only globs `*.partwright.json`, so the palettes dir doesn't
trip it.

**Recipe tests pin figure.md verbatim, grounded in probed values.** Before
writing assertions I probed the actual rig joints for double-biceps
(`abduct 95, elbow 95, twist 90`) and ballet fifth (`abduct 150, elbow 70,
twist 90`) — fists beside the head / hands above the crown curling toward
the midline without crossing it. The chair-sit recipe was already pinned by
the existing sitting-pose test; rather than duplicate it, the new block adds
the two arm recipes plus a generalized lunge sweep (`flex N + knee N` keeps
the shin vertical for N up to 85° — exactly the configuration the old
cross-product knee hinge distorted as flex grew).

**Verification**: end-to-end bake of the karate figure against a live dev
server using `--palette-from-existing` + `--max-genus 3` +
`--require-labels skin,eyes,iris,pupil,headband,belt` → all 11 labels
painted, exit 0; a cube baked with impossible gates → both GATE FAIL
diagnostics, exit 1, no entry written. 61/61 figure unit tests green.
