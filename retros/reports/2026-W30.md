- **A rework batch that has every subagent return a structured 4-Ls retro, then hand-aggregates them with explicit vote counts**, is exactly this process's own method applied one level down (`2026-07-20-catalog-refresh-agent-feedback.md`, 16 agents → one file). It made this week's highest-signal cluster arrive pre-triaged. Worth naming as a pattern to repeat whenever a batch of near-identical rework fans out across many subagents.
- **`model:preview`'s visual loop keeps catching what stats can't**: the corduroy-terracing defect in `smoothWeld` and multiple thin-shell rounding failures were both caught by *looking*, not by a metric — the doctrine CLAUDE.md already states is still paying for itself every time it's used.
- **Parallel `model-sculpt`/subagent delegation for multi-round visual iteration** kept 5 showcase-model render loops (up to 65 tool calls each) and a 16-agent catalog rework batch entirely out of the main/reviewing agent's context.
- **Reading the existing tests before writing code, and tracing new behavior through every existing assertion by hand**, proved backward-compat *before* the edit on the 3MF packing-strategy change and landed green on the first run — a repeat of a pattern already praised in prior reports.
- **Pure-module-first splits** (`layout.ts`/`sharedParams.ts` for the assembly view, `partTree.ts` for part grouping) let the trickiest logic get full vitest coverage before any DOM/Three.js was touched, and both held up through the rest of the feature without regressing.
- **Asking the design question before polishing** (read-only overview vs. full editing, for the Assembly grid view) collapsed four separate bug reports into one coherent model instead of hardening the wrong shape.
- **The PLAYBOOK ratchet continues to compound across new geometry classes** in the inverse-CAD framework — armor went 13-for-13 with "no new tactics needed," and a never-seen 225k-tri internet mesh (3DBenchy) converged in 6 attempts using the armor-era recipe verbatim.
- **`work-reviewer` earned its keep again**, catching a real drag-regression in the part-grouping PR that the author's own e2e coverage had missed.
